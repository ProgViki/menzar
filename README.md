ChatApp Mobile Application

A modern real-time mobile chatting application built with Kotlin and Android Studio, designed with scalability, performance, and clean architecture in mind. The application delivers seamless one-on-one and group messaging, media sharing, notifications, authentication, and modern UI/UX experiences using the latest Android development technologies.

---

🚀 Features
🔐 Secure Authentication (Email, Google, OTP, etc.)
💬 Real-Time Messaging
👥 One-to-One & Group Chats
📸 Image, Video & File Sharing
🎙️ Voice Notes & Audio Messages
🔔 Push Notifications
🟢 Online / Offline Presence
✍️ Typing Indicators
❤️ Message Reactions & Emojis
📌 Pinned Messages
🌙 Dark & Light Theme Support
🔎 Chat Search Functionality
☁️ Cloud Storage Integration
📱 Responsive & Modern UI
🔒 End-to-End Secure Communication
📶 Offline Caching & Synchronization
